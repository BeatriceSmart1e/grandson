# Grandson Website

This folder contains a one-page promotional website for Grandson.

It is a plain static site designed to be pushed to its own repository and hosted
with GitHub Pages. The download button is currently a placeholder and does not
link to a packaged installer yet.

## Preview Locally

Open `index.html` directly in a browser.

There is no build step, dependency install, Docker container, backend, or local
server required for the current page.

## Host With GitHub Pages

1. Push this folder's contents to a GitHub repository.
2. In the repository, open `Settings` -> `Pages`.
3. Set the source to deploy from the main branch.
4. Choose the folder that contains `index.html`. If this website is the whole
   repo, choose the repository root.
5. Save the Pages settings and wait for GitHub to publish the site.

## Files

- `index.html`: page content and structure
- `styles.css`: responsive visual design
- `docs/`: documentation for this website
